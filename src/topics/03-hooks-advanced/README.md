# 03 Hooks Advanced

Advanced hooks patterns including useReducer and useContext.

## Contents

### Puzzle01: useReducer Basic
- **Topic**: Complex state management
- **Key Concept**: reducer function + dispatch pattern
- **When to use**: Multiple sub-values, complex transitions

### Puzzle02: useContext + useReducer
- **Topic**: Global state without Redux
- **Key Concept**: Combine Context API with useReducer
- **When to use**: App-wide state without external libraries

## Key Takeaways

1. useReducer = useState + action dispatch
2. Context avoids prop drilling
3. Together = simple Redux-like pattern
